CLOUDHOST

DESCRIPTION:
cloudhost is a standalone executable jar that emulates cloud storage on a directory environment using Jetty.

ROUTINES:
mrt-cloudhost-1.0.jar - executable cloudhost jar
cshrun.sh - example run script
cshrunlog.sh - example run script with lo
testrun.sh - example start script cshrun.sh
sslstate.sh - example ssl state request
state.sh - example state request
